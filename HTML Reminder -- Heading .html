HTML Reminder -- Heading

<h1>, <h2>, <h3>, <h4>, <h5>, <h6> elements

Tip: Use <h1> to <h6> elements only for headings. Do not use them just to make text bold or big. Use other tags for that.

This is heading 1 <h1>

This is heading 2 <h2>

This is heading 3 <h3>

This is heading 4 <h4>

This is heading 5 <h5>
This is heading 6 <h6>
